Helpful Vim Commands List

Write and Quit: :wq
Visual Mode: v
Select Lines: shift-V
Copy (Yank): y
Paste: p
Delete Selection: d
Jump to End of Line: $
Jump to Beginning of Line: ^
Jump to Next Word: w
Jump to Previous Word: b
Split Screen: split
Vertically Split Screen: vsplit
Open a File: edit
Quit Without Writing: :q!
Undo: u
Redo: ctrl-y
Search: /, next: n, previous: N
Find and Replace: %s/foo/bar/g
Escape to Shell: !
Set Line Numbers set number
